# Forma by SOE Consulting

HR Recruitment Platform — soeconsulting.ai

## Deploy en Vercel (5 pasos)

### Paso 1 — Sube este código a GitHub
1. Ve a github.com → New repository
2. Nombre: `forma-soe`
3. Privado o público, no importa
4. Click "Create repository"
5. Sube todos estos archivos

### Paso 2 — Conecta con Vercel
1. Ve a vercel.com
2. "Add New Project"
3. Selecciona el repo `forma-soe`
4. Click "Deploy" — sin cambiar nada más

### Paso 3 — Conecta soeconsulting.ai
1. En Vercel → tu proyecto → Settings → Domains
2. Escribe `soeconsulting.ai` → Add
3. Vercel te da 2 registros DNS
4. Ve a Wix → tu dominio → DNS → agrega esos registros

### Listo
Tu app estará en https://soeconsulting.ai
